assignment_ind

Ingresar Empleados

Se puede hacer ingreso de Empleados apretando el boton de "Agregar Empleados" designado de color rojo en la parte superior derecha, donde se debera llenar los campos del formulario con los datos del empleado para despues ser ingresado a la API de "empleados" en el archivo json "db.json".

Actualmente se dispone de la Opciones para Editar/Actualizar y Eliminar Datos que se hayan ingresado previamente

folder

Ingresar Productos

De la misma manera y estilo que se pueden Ingresar empleados tambien se pueden ingresar productos con el boton correspondiente

El "id" es designado de manera automatica de ser necesario el ingreso de un codigo fijo por producto el codigo puede ser modificado para adaptar datos necesarios

Revision testing API externa

Con el requerimiento de probar una API de forma externa a servidores locales se hizo conexion con una API abierta que simplemente entrega detalles de usuarios (actualmente no se dispone un metodo de modificiacion de estos)
